## Usage

- Install dependencies:

  ```bash
  pnpm install
  ```

- Run the development server:

  ```bash
  pnpm dev
  ```

- Build for production:

  ```bash
  pnpm build
  pnpm start
  ```

The site typically runs at `http://localhost:3000` when the dev server is started.
